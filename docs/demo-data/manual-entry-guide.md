# Manual product entry guide (no CSV / no paid plan needed)

CSV import is blocked on trial accounts, so this is the same 10 products from `bloom-home-products.csv`, laid out for manual entry via **Admin → Products → Add product**.

For images: Admin's "Add product" media picker only accepts uploaded files, not a pasted URL (that's a CSV-only shortcut). Open each image URL below in a new tab, right-click → **Save image as...**, then upload it when creating the product.

For each product: fill Title/Description/Price, add the variant option(s) with their values, set inventory per variant, add the tag (for the collection it belongs to), upload the image, set status to **Active**.
